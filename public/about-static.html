Museums Web Service

This service provide a system for booking reservations for the museums in Florence.

List Museums

GET all museums available in Florence.

Example Path

XML: https://florence-museums.herokuapp.com/museums.xml?page={page_num}

Params

page (Optional) : The page number to fetch. Defaults to first page.

Auth

This service does not require authentication.

Sample Response

<museums> <museum> <name>VILLA_MEDICEA_DI_CAREGGI</name> <uri>http://localhost:3000/museums/VILLA_MEDICEA_DI_CAREGGI</uri> </museum> <museum> <name>MUSEO_DELL'OPERA_DEL_DUOMO</name> <uri>http://localhost:3000/museums/MUSEO_DELL'OPERA_DEL_DUOMO</uri> </museum></museums> (lifting)

Show Museum

GET a specific musem by name.

Example Path

XML: https://florence-museums.herokuapp.com/museums/{museum_name}.xml

Params

museum_name : The name of the museum you want to get.

Auth

This service does not require authentication.

Sample Response

<museum> <name>MUSEO DELL'OPERA DEL DUOMO</name> <phone>055 230885</phone> <address>Piazza del Duomo n.9</address> <webLink>www.operaduomo.firenze.it</webLink></museum> (lifting)

Create User

POST a new user.

Example Path

XML: https://florence-museums.herokuapp.com/users.xml?name={user_name}&email={user_email}&password={user_password}&password_confirmation={user_password}

Params

user_name : The name of the new user.

user_email : The email of the new user.

user_password : The password of the new user.

Auth

This service does not require authentication.

Sample Response

<notifications><notification> <msg>success</msg><description>user created</description> </notification></notifications>

Show User

GET a specific user.

Example Path

XML: https://florence-museums.herokuapp.com/profile.xml

Params

No parameters required.

Auth

This service does require authentication.

Sample Response

<user> <name>luco</name> <email>baldo@lilik.it</email> <reservations>http://localhost:3000/reservations</reservations></user>

Create Reservation

POST a new reservation.

Example Path

XML: https://florence-museums.herokuapp.com/reservations.xml?museum={museum_name}&date={date_of_visit}&num_of_pipl={number_of_people}

Params

museum_name : The name of the museum you want to visit.

date_of_visit : The date of your visit (dd-mm-yyyy).

number_of_people : The number of visitors.

Auth

This service does require authentication.

Sample Response

<notifications> <notification> <msg>success</msg> <description>reservation created</description> </notification></notifications>

Index Reservation

GET the reservations.

Example Path

XML: https://florence-museums.herokuapp.com/reservations.xml

Params

No parameters required.

Auth

This service does require authentication.

Sample Response

<reservations> <reservation>http://localhost:3000/reservations/2</reservation> <reservation>http://localhost:3000/reservations/3</reservation></reservations>

Show Reservation

GET one reservation.

Example Path

XML: https://florence-museums.herokuapp.com/reservations/{reservation_id}.xml/

Params

reservation_id : The unique identifier of the registration.

Auth

This service does require authentication.

Sample Response

<reservation> <museum>http://localhost:3000/museums/MUSEO NAZIONALE ALINARI DELLA FOTOGRAFIA</museum> <date>22-09-2012</date> <num_of_pipl>4</num_of_pipl> <user>http://localhost:3000/users/2</user></reservation>
